## 💻 Sobre o Projeto

O NanoBox é uma alternativa física e descentralizada aos serviços tradicionais de nuvem (Big Techs), operando no modelo **HaaS (Hardware as a Service)**. Este repositório contém a arquitetura Front-end (Landing Page) do projeto, projetada com foco em conversão, quebra de objeções técnicas e design minimalista (Dark Mode/Glassmorphism).

Este projeto foi desenvolvido como escopo prático e acadêmico para o curso de Análise e Desenvolvimento de Sistemas (Uniube), demonstrando conceitos de UI/UX, componentização e desenvolvimento web moderno.

## ✨ Funcionalidades da Interface

A landing page foi construída seguindo a metodologia de "Atomic Design" e fluxo narrativo:
- **Hero Section:** Apresentação impactante com tipografia forte e chamadas para ação.
- **Seção de Problema & Solução:** Quebra do paradigma atual das nuvens públicas.
- **Como Funciona:** Fluxo passo a passo demonstrando a facilidade *Plug-and-Play*.
- **Ecossistema Mobile:** Mockup de integração com aplicativo de gestão remota.
- **Arquitetura de Segurança:** Destaque para Criptografia Ponta a Ponta, Snapshots Anti-Ransomware e Firewall Ativo.
- **Planos (Pricing):** Apresentação do modelo de negócios por assinatura.
- **FAQ Técnico:** Accordion interativo para quebra de objeções.

## 🛠️ Tecnologias Utilizadas

- **[React](https://reactjs.org/)** - Biblioteca JavaScript para construção de interfaces.
- **[Vite](https://vitejs.dev/)** - Build tool ultrarrápido para projetos web.
- **[Tailwind CSS](https://tailwindcss.com/)** - Framework de utilitários CSS para estilização rápida e responsiva.
- **[Framer Motion](https://www.framer.com/motion/)** - Biblioteca de animações baseada em física (reveal no scroll, stagger).
- **[Lucide React](https://lucide.dev/)** - Pacote de ícones minimalistas.
